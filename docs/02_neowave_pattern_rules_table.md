retracement:** 0.236–0.999 of wave1 length ( >23.6% and <100%).<br>- **Wave5 length:** ≥ 0.382 * wave4 length.<br>- **Alternation:** Check retracement(depth) and complexity: (wave2_depth ≠ wave4_depth within ±10%) and (wave2_complexity ≠ wave4_complexity) (not equal in characteristics). | **Terminal Impulse (Diagonal)**  | 5-wave pattern where waves 2 & 4 do overlap (wedge shape). Often last in a sequence. Waves are typically zigzags internally.                                                                    | - Allow overlap: wave4_end > wave1_end (bullish case) but maintain overall 5-wave count.<br>- **Structure:** Require each of 5 sub-waves to be zigzag/3-wave (for an ending diagonal) rather than clear impulses.<br>- **Wave relationships:** Still enforce wave3 not shortest; wave5 ≥ 0.382 of wave4.<br>- Converging trendlines: wave1 > wave3 > wave5 in length (contracting diagonal), or sometimes expanding form (wave1 < wave3 < wave5 for rare expanding diagonal).<br>- Volume downtrend throughout pattern (often).                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            | - If any supposed sub-wave (1,3,5) is not a zigzag/3-wave, the diagonal interpretation fails.<br>- If wave3 is shortest or no extension present (same as impulse rules) ⇒ *invalid* even as diagonal.<br>- If pattern does not form a clear wedge (e.g., wave5 fails to be smaller in contracting case with overlap) ⇒ likely miscount.<br>- If wave2 or 4 exceeds 100% retracement of prior wave (e.g., wave4 goes beyond start of 3 by too much) ⇒ *not a viable diagonal* (should still respect 50-138% typical range).                                                                                                                                                                                                                                                                                                                                   |
| **Wave 2 Depth vs Wave 4 Depth** | Alternation in correction severity: one deep, one shallow. In a strong extension case, wave2 often sharp, wave4 shallow sideways (or vice versa).                                               | - Compute retracement ratios: r2 = wave2_length / wave1_length, r4 = wave4_length / wave3_length.<br>- **If** r2 > 0.618 (deep wave2) **then expect** r4 < 0.5 (shallow wave4). Conversely, if r2 < 0.5, expect r4 > 0.618.<br>- Flag if both r2 and r4 are both > ~0.618 or both < ~0.236 (i.e., no alternation in depth).                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                | - If wave2 and wave4 both retrace > 60% (both deep) *and* both are similar simple patterns ⇒ violates alternation (less likely impulse) – *issue warning*.<br>- If one of wave2 or wave4 retraces < 15% (extremely shallow) and the other also shallow ⇒ the larger trend might not have been broken (possible running correction or miscount).                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
| **Extension Identification**     | In any impulse, one of waves 1,3,5 must be significantly longer (the extended wave). Typically wave3.                                                                                           | - Measure lengths L1, L3, L5. Identify max Lm and second max Ls.<br>- Check Lm ≥ 1.618 * Ls. If true, flag that wave as extended (e.g. wave3_ext = True if wave3 is longest and ≥1.618x next).<br>- If wave1 is longest and meets 1.618 rule, mark wave1 extended scenario; if wave5 longest ≥1.618x others, wave5 extension.<br>- If no wave meets 1.618 ratio, check exceptions: (wave3 slightly <1.618 of 1, with 5 smaller) or (wave1 ~1.5x wave3 with wave3 relatively small extension).                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              | - If no wave satisfies extension ≥1.618x and neither special case applies ⇒ *impulse invalid* (treat as corrective).<br>- If two waves appear similarly long (within 10%) and much larger than third (e.g., w1 ≈ w3 > w5), no single clear extension – likely a terminal pattern or miscount (flag for manual review).                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
| **Rule of Equality (Impulse)**   | The two non-extended impulse waves tend towards equality or a Fibonacci ratio. Example: if wave3 extends, waves1 ≈ 5.                                                                           | - After finding extended wave, compute ratio of other two waves in price and time.<br>- Ideally ratio ≈ 0.9–1.1 (near equality) or ≈0.618 (one is 61.8% of the other).<br>- Example check: If extended = wave3, test if 0.618 ≤ (wave5_length/wave1_length) ≤ 1.618. If ~1.0, equality rule satisfied. If ~0.618, Fibonacci relation satisfied.<br>- Same for duration: shorter of (T1,T5) at least 0.618 of longer.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       | - Not a strict invalidation, but a diagnostic: If non-extended waves differ hugely (<< 0.5 or >>2.0 ratio), then impulse might be suspect or subdivided.<br>- If extended wave is unclear and waves1 and5 are not fib-related at all, impulse may need reconsideration (though equality is a guideline, not hard rule).                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |

Possibly wave5 is extending or pattern was something else.<br>- If line never breaks and price continues impulse direction, obviously impulse wasn’t done (invalid end).                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| **Zigzag Pattern**               | A sharp 3-wave correction (5-3-5 internal). Wave B is shallow, wave C often equals or exceeds A.                                                                                                | - Identify 3 swings labeled A, B, C with net move against trend.<br>- **Wave A:** impulsive or leading diagonal structure (verify 5 sub-waves). Often  WaveA_retracement_of_prev_impulse < 0.618 (it shouldn’t fully retrace prior trend).<br>- **Wave B:** retraces 10%–61.8% of A (use ≤61.8% max). Typically category: shallow (e.g., 0.382).<br>- **Wave C:** impulsive 5-wave down (or up). Ensure C_length ≥ 0.618 * A_length.<br>- Classify zigzag type by C:<ul><li>if C < 0.618*A: truncated zigzag.</li><li>if 0.618*A ≤ C ≤ 1.618*A: normal zigzag.</li><li>if C > 1.618*A: elongated zigzag.</li></ul><br>- **X-wave check:** If this zigzag is part of combo and channels neatly, expect an X after it.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       | - If wave B > 61.8% of A ⇒ *Not a zigzag* (likely a flat).<br>- If wave A or C is not a 5-wave structure ⇒ invalid zigzag labeling.<br>- If wave C < 0.618 A (truncated), require confirmation: next move must retrace ≥ 0.81 of zigzag; if not, truncation count invalid (zigzag likely part of bigger pattern).<br>- If elongated (C > 1.618A), require post-C rebound > 61.8% of C quickly to confirm correction. Failure to do so means what was thought a zigzag C might be extending impulse (count invalid as zigzag).                                                                                                                                                                                                                                                                                                                                |
| **Flat Pattern**                 | A sideways 3-wave correction (3-3-5 structure). Wave B is large (≥ 61.8% of A), wave C is smaller than or equal to prior swing range often. Several sub-types: regular, expanded, running.      | - Identify 3 swings A, B, C roughly sideways net motion.<br>- **Wave A:** 3-wave structure (often zigzag shape). No specific fib min, but typically 15–50% retrace of preceding move.<br>- **Wave B:** ≥ 61.8% of A. Categorize:<ul><li>61.8–80%: **weak B flat**.</li><li>81–100%: **normal B flat**.</li><li>>100%: **strong B (expanded flat)**.</li></ul>Check if B goes beyond start of A ( >100%) = expanded scenario.<br>- **Wave C:** 5-wave impulse/ending diag. Must be ≥ 0.382 * A. Typical outcomes by B type:<ul><li>Normal B (≈100% A): C usually ≈100%-138% of B (makes new extreme beyond B-start).</li><li>Weak B (≤80% A): C likely exceeds B, often 100-138% of B (to compensate).</li><li>Strong B (>100% A): consider **expanded flat**. If B 101–123% A, C likely exceeds A’s start (irregular flat). If B very large (>138% A), C will probably fail to reach A’s start (running flat).</li></ul>Use horizontal lines at A’s start/end to judge C’s reach.<br>- Determine subtype:<ul><li>**Regular flat:** B ≈ A (81-100%), C ≈ A (often slightly > start of A).</li><li>**Expanded flat:** B > A, C goes beyond A’s start (at least slightly).</li><li>**Running flat:** B > A, C < A’s start (C fails to make new extreme).</li><li>**Failure flat:** C fails to reach B’s start (could be c-failure if B normal, or double-failure if B weak).</li></ul><br>- **Internal checks:** wave B should be a 3-wave structure itself; wave C a 5-wave. | - If wave B < 61.8% of A ⇒ *Not a flat* (likely zigzag).<br>- If wave C < 0.382 of A ⇒ pattern didn’t correct enough (invalid flat labeling).<br>- If both A and B are zigzags and B is deep, but wave C came out as 3-wave, not impulse, then likely the pattern is part of combo (flat incomplete) – invalid flat by itself.<br>- Running flat logic: If B > 1.382 A and C does exceed A’s start significantly, that contradicts Neely’s note (virtually no chance) – likely miscount. If labeled running flat, ensure next move continues in trend direction (since running flat is basically trend continuation). If price instead reverses after a running flat, the interpretation might be wrong.<br>- Perfect channel flat (A-C parallel, 0-B parallel) signals likely an X-wave to follow – the flat alone might be incomplete in larger structure. |
| **Triangle (Contracting)**       | 5-wave coiling pattern (A–E) where each wave is smaller. Waves all corrective. A longest, E smallest.                                                                                           | - Identify 5 swings A,B,C,D,E with overall converging highs/lows.<br>- **Wave A:** largest segment. **Wave C:** ~61.8% of A. **Wave E:** 38.2%–99% of C (E at least 0.382*C but not exceeding C’s start).<br>- Check each wave is a “3” internally.<br>- Trendlines from A to C and B to D should converge to an apex beyond E.<br>- Volume typically dwindling each wave.<br>- Use **Reverse Alternation** check: If B was notably small (~<=0.382 of A) then expect D > C (D retraces much of C) causing nearly parallel sides. Otherwise, standard: B ~0.5-0.786 of A, D ~0.5-0.786 of C, lines converge normally.<br>- Ensure no wave overshoots where it shouldn’t: A sets initial range, C < A, E < C. B < A, D < C in price extremes.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                               | - If any wave (C or E) exceeds the length of a prior same-direction wave (C > A or E > C) ⇒ *not contracting triangle*.<br>- If E < 0.382 of C (too short), triangle likely incomplete or misidentified (could become a 7-wave pattern).<br>- If price fails to stay within converging trendlines (e.g., D breaks above line through A-C) ⇒ triangle count invalid at that time.<br>- If all 5 points A,B,C,D,E lie exactly on two trendlines (each side touched 3 times) ⇒ pattern likely *too perfect* – expect additional waves (invalidate as final triangle).<br>- If volume doesn’t decline through A→E, re-check pattern (triangles should show weakening volume).                                                                                                                                                                                    |
| **Triangle (Expanding)**         | 5-wave diverging triangle. Each wave larger. A smallest, waves fan out.                                                                                                                         | - Identify A–E with price extremes widening each wave.<br>- **Wave A:** smallest. **Wave C:** > 1.0 * A (at least 101% – must go beyond A’s end), typically up to 1.618*A. **Wave E:** > 1.0 * C (E extends beyond C’s end), typically ≤1.618*C (unless blow-off case).<br>- Trendlines from A->C and B->D diverge.<br>- Volume might increase or be erratic (but often still lower on E than A if trend exhausting).<br>- Reverse alternation: If B was extremely large relative to A (e.g., 1.38x or more), then D will be smaller relative to C (D might retrace only ~0.4 of C) causing sides to be more parallel than fully diverging.<br>- Internals: all waves corrective 3s.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       | - If C ≤ A (doesn’t go beyond A’s extremum) ⇒ *not expanding triangle*.<br>- If E ≤ C (fails to extend beyond C’s extreme) ⇒ pattern incomplete or mis-labeled (maybe only a contracting or neutral triangle).<br>- If wave E >> wave C (exceeds 1.618 by a lot), likely not a controlled triangle – possibly last wave blow-off. Neely allows up to ~2.618*C in rare blow-off, but beyond that count should be reconsidered (maybe E is part of new impulse).<br>- If triangle expands but then volume drops strongly on E (contrary), treat with caution (market might have treated E as climax, watch for reversal to confirm pattern end).                                                                                                                                                                                                               |
| **Triangle (Neutral)**           | 5-wave triangle where wave C is the longest, A and E are about equal and moderately sized. Trendlines roughly parallel.                                                                         | - Identify 5 waves A–E without a clear converging or diverging trend – likely parallel boundary lines.<br>- **Wave C:** longest wave of the five.<br>- **Wave A ≈ Wave E:** both should be ≥ 0.382 of C, typically 0.618–0.72 of C length. A and E often similar length (within ~10%).<br>- Waves B and D then are the smaller alternating waves (usually B < A, D < C, and roughly similar to each other).<br>- Trendlines: A-C and B-D lines end up nearly parallel (slightly contracting or expanding). Volume usually flat to slightly decreasing.<br>- Check reverse alternation: If B is shorter in time than A (allowed if B smaller in price), often D will also adjust accordingly (could cause slight tilt in one of the trendlines). Neutral can handle some flexibility in B, D without changing type.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         | - If wave C is not the longest (e.g., A or E match or exceed C) ⇒ *not a neutral triangle*.<br>- If waves A and E are nowhere near equal (say E is half of A or twice A) and not ~0.618 of C either ⇒ pattern likely misidentified – might be contracting or expanding instead.<br>- If trendlines are not parallel (significantly converging or diverging), then triangle is likely contracting or expanding, not neutral. Neutral should appear almost rectangular in outline (slight slope okay).<br>- If volume pattern is strongly one-directional (like steady decline as in contracting or steady rise as in expanding), then it might not be a truly neutral scenario – re-check lengths.                                                                                                                                                            |
| **Triangle B/D Alternation**     | B and D waves in triangles should be similar in behavior or show “alternation.” Standard: if B is deep, D is shallow, etc. Reverse alternation flips this.                                      | - Compute retracements: b_rel = B_length/A_length, d_rel = D_length/C_length (taking absolute lengths).<br>- **Standard alternation:** if b_rel ≥ 0.618 (B relatively large), expect d_rel ≤ 0.618 (D smaller). If b_rel ~0.38, expect d_rel ~0.8 (D bigger than typical) – though B that small triggers reverse alt scenario.<br>- **Reverse alt condition:** If B < 0.5 A in contracting triangle, enforce D > C (D retrace >100% of C amplitude) to fulfill reverse alternation. If B > 1.0 A in expanding, enforce D much smaller (perhaps D ~0.5 or less of C).<br>- Program logic: determine expected D_behavior from B. If actual D deviates, decide if that’s reverse alt or miscount.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             | - If triangle sides don’t match expected geometry (e.g., contracting triangle but lines not converging due to an odd B or D), check if reverse alternation case. If not explained by reverse alt, then triangle classification might be wrong.<br>- If reverse alternation scenario yields trendlines that are parallel or wrong orientation (e.g. contracting triangle with lines parallel or diverging without valid reason), then pattern might not be a triangle or might be neutral mislabeled as contracting.<br>- Essentially, if B & D behavior violates both standard and reverse rules (e.g., B is shallow but D also shallow relative to respective waves, leaving a gap), then triangle count is suspect.                                                                                                                                        |
| **Combination (Double Three)**   | Two corrective patterns connected by an X-wave. E.g. W-X-Y where W and Y are zigzag/flat/triangle. Allows longer, more complex correction.                                                      | - Identify sequence of 3-3 (or 5-3-3 etc) that doesn’t fit a single pattern but can be split:<br>Find a smaller connecting wave **X** that is clearly corrective and relatively brief between two larger corrections **W** and **Y**.<br>- Ensure **W** (waves before X) forms a valid single correction (zigzag, flat, or triangle) and **Y** (waves after X) forms another valid correction pattern.<br>- X-wave usually a zigzag or flat that retraces ≤ 0.618 of W (often ~0.50). X typically smaller in price & time than W or Y.<br>- Overall, the two patterns W and Y can be of different types (commonly zigzag + flat, flat + zigzag, or either + triangle at end).<br>- Check that total wave count fits W(3 or 5 waves) + X(3 waves) + Y(3 or 5 waves) = 7 or 11 segments total.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                               | - If no sensible break point for X is found (i.e., the correction can’t be cleanly split into two distinct patterns), then don’t force a combo – it might be a single complex pattern instead.<br>- If either W or Y is not a valid pattern on its own (fails its rules) ⇒ invalid double-three count.<br>- If X is too large (e.g., retraces almost all of W or lasts as long as W) ⇒ it’s not a connecting blip, likely the whole thing was one pattern or needed different degree labeling.<br>- If more than one X-wave (i.e., trying to chain 3 patterns) – then it’s actually a triple three, not a double (handle separately).                                                                                                                                                                                                                        |
| **Triple Three**                 | Three corrections linked by two X waves (W-X-Y-X-Z). Very rare max complexity.                                                                                                                  | - Identify two X waves splitting three segments W, Y, Z.<br>- Each of W, Y, Z must be a valid corrective pattern (zigzag/flat/triangle). Possibly one could be a diametric in rare case, but typically simpler parts.<br>- X waves should be small relative connectors (same criteria: shallow retracements of preceding segment, shorter duration).<br>- Total wave count: 3 patterns + 2 X’s = typically 11 segments (if each part W,Y,Z is 3-wave basic structure, overall 11 swings; more if triangle part counted by sub-waves).                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      | - If needed to assume a second X but patterns don’t clearly support so many parts, caution: quadruple combinations are not allowed – drop count if requiring 3rd X.<br>- If any of the three component patterns fails its own rules, triple-three interpretation fails.<br>- The more complex, the harder to confirm. If any simpler interpretation exists that fits, triple-three should not be first choice. It’s invalid if likely a simpler diametric or symmetrical could explain the structure more parsimoniously (subjective, but algorithm can check uniformity to decide between triple combo vs diametric).                                                                                                                                                                                                                                       |
| **Diametric (7-wave pattern)**   | A single 7-leg corrective pattern (A-G) where waves expand then contract symmetrically. Waves are similar in time, moderate differences in price.                                               | - Detect 7 swings A through G with a shape that first diverges (A to C perhaps increasing) and then converges (E to G decreasing). Wave D is a middle “axis”.<br>- Measure time of each wave: T(A)...T(G). Expect most T’s to be roughly equal (within ~10-20%), with at most one or two slightly longer/shorter.<br>- Price: measure wave magnitudes. Typically A < C < E (expansion), then E ~ largest, then G > C? Actually, diametric: first 4 waves (A–D) might show net expansion or contraction bias that reverses in next 3 (E–G). So check if wave lengths follow a pattern like A < B < C ~ D ~ E > F > G (just an example pattern). Specifically: by wave D, pattern flips bias, G ends near a size comparable to A perhaps.<br>- All legs are corrective 3-wave internally (like mini zigzags/flats).<br>- The shape often superficially resembles a stretched-out triangle or bow. Use Neely’s note: if you see a 7-wave formation where each segment is same degree and neither a simple combo nor triangle, suspect diametric.                                                                                                                                                                                                                                                                                                                                                                                                                              | - If 7-wave sequence has large disparity in wave durations (say one wave takes 3x another) ⇒ not a diametric (since diametric requires time similarity).<br>- If price progression doesn’t show a single peak in middle (wave E often a zenith or nadir) followed by symmetry, might not be diametric.<br>- If one can identify two X-waves inside it, then it might actually be a double-three, not a diametric. Diametric is a single entity; if forced to split, do so and diametric invalid.<br>- If pattern channels too perfectly or doesn’t reverse expansion/contraction in second half, it could be just a weird triangle – diametric invalid if it fits simpler category upon closer check.                                                                                                                                                        |
| **Symmetrical (9-wave pattern)** | A single 9-leg pattern (A–I) where waves are extremely uniform in length and duration. Market in equilibrium, minimal variance.                                                                 | - Detect 9 swings of roughly equal size and time. Compute max deviation in wave price and time.<br>- Should find differences ≤ 10% among waves for symmetrical. For example, each wave maybe 5-6 bars long, and price swing ~similar magnitude.<br>- All waves corrective (3-wave internals). No net trend – pattern is very flat overall.<br>- Usually only identified if clearly not a triple combo (no obvious X connectors) and too many waves for triangle. If everything looks *too* consistent over many swings, symmetrical likely.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                | - If wave lengths or durations vary more than ~10% (some wave clearly larger or longer) ⇒ violates symmetry – not a symmetrical (maybe diametric or combo).<br>- If count is 9 but one can see two smaller X waves and 3 subpatterns, then it’s a triple three, not a symmetrical – symmetrical invalid.<br>- If overall pattern has a slope or trend, it might not be a true symmetrical (which should be sideways). If it trends, might be part of larger pattern, invalid as standalone symmetrical.                                                                                                                                                                                                                                                                                                                                                      |
| **Rule of Similarity/Balance**   | Adjacent same-degree waves must share similar scale in price or time (≥ 33% of each other). Ensures proportional labeling.                                                                      | - For every pair of neighboring waves (esp. impulse vs correction, or two legs of correction), check:<br>`min(price_wave1, price_wave2) / max(price_wave1, price_wave2) ≥ 0.33` **OR** `min(time_wave1, time_wave2) / max(time_wave1, time_wave2) ≥ 0.33`.<br>- If not, likely degree mis-match: smaller wave is of lower degree. Program uses this to prevent counting a tiny blip as full wave if it’s disproportionate.<br>- Example: If a correction after an impulse is only 10% of impulse’s length and duration, it’s probably not same degree – perhaps correction is incomplete or part of lesser degree.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         | - If a wave fails both price and time 1/3 tests relative to adjacent wave ⇒ *cannot be same degree*. The algorithm should then not label them consecutively; likely it means what was thought two waves are one, or one is part of substructure.<br>- If enforcing this globally: if you find your count has any adjacent waves violating this, the count is wrong degree-wise. The program would adjust (e.g., merge a too-small “wave” with its neighbor).<br>- Essentially, violation triggers a need to redraw or relabel degrees until all adjacent waves pass similarity test.                                                                                                                                                                                                                                                                         |
| **Time-Proportional Limits**     | Corrections typically last at least a certain fraction of the preceding impulse’s time. (Neely’s time rules: Flats/Zigzags have largest time differences, longer patterns smaller differences). | - Impose a *heuristic* minimum: e.g. a correction (single zigzag/flat) should consume ≥ 0.33 of the time of the impulse it corrects (often more). If less, likely incomplete or part of larger correction.<br>- Use NEoWave time logic: For a given A-B-C:<ul><li>If T_B ≈ T_A, expect T_C ≈ T_A + T_B.</li><li>If T_B >> T_A, expect T_C ≈ (T_A+T_B)/2.</li><li>Wave B never takes less time than A in flats/zigzags (enforce B_time ≥ A_time).</li></ul>Check actual times and see which pattern fits. If none, pattern might be wrong type.<br>- For triangle legs: wave C usually longer than A; wave B can be shorter than A *only* if B_price < A_price (allowed with reverse alt). Incorporate that condition.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      | - If a supposed correction’s total time is extremely short relative to prior wave (say < 0.2 of prior impulse’s time), likely miscounted degree – mark invalid as standalone (it might be just part of ongoing correction).<br>- If wave B of a flat/zigzag is timed shorter than wave A despite being large in price (contradicts rule), then the time dimension violates expectations – perhaps not labeled correctly (maybe that B was an X of prior smaller pattern?).<br>- If time relationships in a proposed pattern grossly deviate from Neely’s guidelines (like wave C taking far more or far less time than predicted by A and B’s times), consider that pattern interpretation low-confidence or invalid. Possibly pattern is morphing into complex or of different degree.                                                                      |
| **Volume Signature**             | Typical volume patterns: Impulse – rising volume into wave3【*】, lower on wave5; Corrections – declining volume except possible spike on capitulation. Triangles – volume contracting each wave. | - For an impulse count, check volume at wave peaks: `Vol3 > Vol1` and `Vol3 > Vol5` expected (especially if wave3 extended). If `Vol5 > Vol3` significantly, flag – maybe blow-off or miscount (maybe that “wave5” was actually wave3 of bigger extension).<br>- For a correction, verify volume overall reduces. E.g., measure average volume per segment: A vs C in zigzag (often `VolA >= VolC` or similar); in flats, `VolB` might be high (if expanded flat mania), but `VolC` should drop off or at most equal A.<br>- Triangle: ensure `Vol(A) > Vol(C) > Vol(E)` (diminishing) ideally. Program can do linear regression on volume across waves, should be negative slope for contracting/neutral triangles. Expanding triangle might not follow this strictly, but often volume isn’t surging hugely either until possibly wave E blow-off.<br>- Use volume ratio thresholds: e.g., if supposed impulse has wave5 volume 2x wave3, likely not a normal impulse (maybe a 3rd extension or new trend start).                                                                                                                                                                                                                                                                                                                                                                                                                                                        | - No single volume pattern invalidates a count (volume is auxiliary), but certain divergences strongly suggest re-evaluation:<br>- If volume increases through each impulse wave (no peak at 3) – maybe the move isn’t an impulse but part of larger sequence or an accelerating diagonal (which are rare). If volume pattern mismatches, lower confidence in count.<br>- If a triangle doesn’t show declining volume, question it – maybe it’s a series of zigzags instead.<br>- If a claimed zigzag’s wave C has much higher volume than wave A (exceeding even prior impulse volumes), consider it might be an impulse wave starting, not just a zigzag ending.<br>- The program can mark counts that contradict expected volume behavior as suspect but not outright invalid – volume is a softer rule.                                                  |
